Account Setup

My Role

UI/UX Design | Frontend Development

Team

Derek Dorrance - Software Dev

Timeline

3 Months - Launched in February 2025

Overview

Users felt overwhelmed during sign-up, leading to incorrect or incomplete account setups. This resulted in wasted support time and lower trial conversion rates.

I led the design and frontend development of a self-guided account setup walkthrough. The goal was to streamline onboarding with a simple, intuitive process, allowing users to input their information quickly and focus on learning the system effectively.

02

Problem

System was lacking a simple walkthrough to help new users properly set up account.

03

Research

My Ideation

Minimal amount of steps with a simple interface to increase user completion.

More informative information when setting up certain sections to ensure proper setup.

Easy way to see the steps and navigate though them.

04

Highlights

Simple step by step guide

A seamless setup experience to engage users to set them up for success with using the system.

Profile Creation

Creating profile and setting up branding from the get go to ensure they have branding setup when using marketing features after setup.

Intuitive team setup

Creating a simple way for users to add and understand the structure of their team members in our system.

Helpful Lead source setup

A section to help users set up their lead sources to set them up in getting leads into the system from the jump.

05

Conclusion

The introduction of the self-guided account setup walkthrough significantly improved the onboarding experience for new users. By simplifying the setup process with a minimal step-by-step guide, users were able to navigate their initial account configuration with ease, reducing feelings of overwhelm. The intuitive team setup and lead source configuration ensured that users could efficiently structure their accounts and start generating leads immediately.

As a result, we saw a reduction in support requests related to the simple setup steps, allowing our support team to focus on more strategic user needs rather than manual setup tasks. This project not only enhanced user engagement but also optimized internal resources, making onboarding more scalable.
